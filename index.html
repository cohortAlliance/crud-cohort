Cohort PHP Challenge

January 11, 2016

Cohort PHP Challenge demonstrates retrieving data from a table called people in a Cloud 9 database called crud. We use SELECT and ECHO to perform and display our data. We use three different forms of PHP to do this:

  1. MySQLi procedural (MySQLi p),
  2. MySQLi object oriented (MySQLi oo),
  3. Challenge (PDO)
